# X-TechInsight — 用户手册

> 本文档教你 **怎么用** 这个平台：从打开网页到上传文档、提问、看懂回答。  
> 零基础按步骤操作即可。

## 二、界面说明（三栏布局）

```
┌──────────────┬────────────────────────────┬──────────────────┐
│  左：对话列表  │  中：聊天区                  │  右：配置面板       │
│              │                            │                  │
│  + 新建对话   │  消息历史 + 输入框            │  领域标签          │
│  对话 1      │                            │  知识库挂载         │
│  对话 2      │                            │  上传文档           │
└──────────────┴────────────────────────────┴──────────────────┘
```

| 区域 | 作用 |
|------|------|
| **左侧** | 管理多个对话，点击切换 |
| **中间** | 提问、看 AI 流式回答、看引用来源 |
| **右侧** | 设置领域标签、上传文档、勾选挂载到当前对话 |

## 三、第一次使用（推荐流程）

按下面 **5 步** 走一遍，就是完整演示流程：

### 第 1 步：新建对话

1. 点击左侧 **「+ 新建对话」**
2. 填写 **对话名称**（如「技术栈讨论」）
3. 选择或输入 **领域标签**（如「后端」）
   - 快捷选项：后端、金融、云计算、前端、general
   - 也可自定义，例如「医疗」「法律」
4. 点击 **「创建」**

### 第 2 步：上传文档

1. 确保已选中一个对话（右侧配置面板可见）
2. 点击 **「上传文档 (.pdf / .md / .txt)」**
3. 选择本地文件

**限制：**

| 规则 | 说明 |
|------|------|
| 格式 | 仅 `.pdf`、`.md`、`.txt` |
| 大小 | 单文件 ≤ **10MB** |
| PDF | 需能选中文字；扫描版、加密 PDF 不支持 |

4. 上传后显示「正在上传并后台解析，请稍候…」
5. 等待 **10～30 秒**（大文件更久），文档会出现在下方 **全局资源池** 列表

> 上传时使用的 **领域标签 = 当前对话的标签**。文档入库时会带上这个 tag，问答时按 tag 检索。

### 第 3 步：挂载文档到当前对话

1. 在右侧 **知识库挂载** 列表中，**勾选** 要用于本对话的文档
2. 未勾选的文档：**不会** 参与本对话的领域冲突检测（见下文）

**说明：** 所有上传的文档进入 **全局资源池**；勾选 = 挂载到 **当前对话**。  
后端检索目前主要按 **对话的领域标签** 过滤，挂载勾选主要用于 UI 管理与冲突提醒。

### 第 4 步：提问

1. 在中间底部输入框输入问题，例如：
   - 「项目用了哪些技术栈？」
   - 「FastAPI 有哪些核心 API？」
2. 按 **Enter** 发送（Shift + Enter 可换行，若支持）
3. 观察 **流式输出**：回答会逐字出现

### 第 5 步：看懂回答

AI 回答分两部分：

| 部分 | 说明 |
|------|------|
| **正文** | Markdown 格式，代码块有高亮 |
| **引用来源** | 回答下方列出 `[Source N]` 对应的文件名、页码、片段摘要 |

**两种典型回答：**

| 回答 | 含义 | 你该做什么 |
|------|------|------------|
| 正常回答 + `[Source 1]` 等 | 检索到了相关文档 | 可点来源核对原文 |
| 「根据现有资料，我无法回答这个问题。」 | 资料不足或未上传/未入库 | 检查是否上传、是否等够解析时间、问题是否与文档相关 |

## 四、领域标签怎么用

### 什么是领域标签？

每个对话有一个 **tag**（领域标签），例如「后端」「金融」。  
系统在向量检索时会 **优先只查同一 tag 的文档片段**，避免金融文档回答后端问题。

### 创建时怎么选？

- 讨论 Java / FastAPI → 选 **「后端」**
- 讨论财报 / 风控 → 选 **「金融」**
- 不确定 → 选 **general**

### 创建后怎么改？

1. 在右侧 **领域标签** 输入框修改
2. 点击 **「保存标签」**
3. 之后的新问答会按 **新 tag** 检索

---

## 五、领域冲突提醒（红色提示）

当你 **勾选挂载** 的文档里，有文档的 tag 与 **当前对话的 tag 不一致** 时，右侧会出现 **红色「领域冲突提醒」**。

**含义：** 你把不同领域的文档挂到了同一对话，检索可能混入噪音。

**建议处理：**

1. **取消勾选** 标签不匹配的文档，或  
2. **更换对话 tag** 与主要文档一致，或  
3. **新建对话** 专门讨论该领域

> 若文档 **未勾选挂载**，不会触发此提醒。

## 六、多对话怎么用

| 场景 | 做法 |
|------|------|
| 同时讨论两个项目 | 建两个对话，各自设不同 tag |
| 切换话题 | 左侧点击另一个对话 |
| 历史记录 | 切换回某对话，中间区自动加载历史消息 |

每个对话的 **消息记录独立** 保存在服务器 SQLite 中（Render 免费版重启可能丢失，演示时注意）。

## 七、常见问题（使用层面）

### Q1：页面空白或一直加载失败

1. 等 30～60 秒（后端唤醒）
2. `Cmd + Shift + R` 强制刷新
3. 再点「新建对话」

### Q2：上传成功但问答仍说「无法回答」

| 可能原因 | 处理 |
|----------|------|
| 解析未完成 | 再等 10～30 秒后重问 |
| 问题与文档无关 | 换与文档内容相关的问题 |
| tag 不匹配 | 确认对话 tag 与文档上传时的 tag 一致 |
| 公网数据被清空 | Render 免费版重启可能丢库，需重新上传 |

### Q3：PDF 上传报错

- 加密 PDF → 换未加密版本或导出为 `.md`
- 扫描版 PDF → 先 OCR 转成可选中文字的 PDF 或 `.txt`

### Q4：能挂多个文档吗？

可以。全局资源池可有多份文档；当前对话可勾选多份挂载。  
问答时系统从 **同 tag** 的片段中检索 Top-K 条最相关内容。

### Q5：需要登录吗？

**不需要。** 当前版本无注册登录，打开网址即可使用。  
适合个人演示；若长期公开给陌生人，建议未来加登录（见 `01-设计思路与架构.md` 二期规划）。

## 八、本地使用补充（Mac 终端）

若你在自己电脑上跑项目：

### 启动后端

```bash
cd ~/Desktop/X-TechInsight/backend
source venv/bin/activate
# 国内可能需要代理
export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890
python main.py
```

### 启动前端（新开一个终端窗口）

```bash
cd ~/Desktop/X-TechInsight/frontend
npm run dev
```

浏览器打开 http://localhost:5173
