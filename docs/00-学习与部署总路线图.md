## 二、推荐免费部署方案（已为你选定）

| 组件 | 平台 | 免费网址示例 |
|------|------|--------------|
| 前端 React | **Render Static Site** | `https://x-techinsight-web.onrender.com` |
| 后端 FastAPI | **Render Web Service** | `https://x-techinsight.onrender.com` |

**为什么推荐这套（而不是只用 IP）：**

- 注册即可用，不用自己买服务器、配防火墙
- 自动给 **HTTPS 网址**（比 `http://IP:端口` 更适合写进简历）
- 和你项目技术栈（Vite + FastAPI）匹配

**需要你知道的限制（面试也能讲）：**

- Render 免费档 **15 分钟无访问会休眠**，第一次打开要等 ~30 秒唤醒
- 免费档 **磁盘不永久**（SQLite / ChromaDB 重启可能丢数据）→ 演示够用，生产要换付费盘或云数据库
- OpenAI Key 在 Render 后台配置，**不要**写进代码或 GitHub
